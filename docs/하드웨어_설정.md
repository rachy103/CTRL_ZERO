# 하드웨어 설정

이 문서는 CTRL_ZERO를 실제 차량 하드웨어에 연결할 때 확인할 항목을 정리합니다.

## 카메라

`main.py` 상단에서 카메라 번호를 바꿉니다.

```python
CAMERA_INDEX = 1
CAMERA_BACKEND = "dshow"
```

Windows에서는 `dshow`가 Logitech 계열 USB 카메라에서 안정적인 경우가 많습니다. 카메라가 열리지 않으면 `CAMERA_INDEX`를 `0`, `1`, `2` 순서로 바꿔 보세요.

## Arduino 직렬 포트

연결된 포트 목록을 확인합니다.

```powershell
python main.py --list-ports
```

자동 감지를 쓰려면:

```python
ARDUINO_PORT = "auto"
```

명시적으로 지정하려면:

```python
ARDUINO_PORT = "COM3"
```

## Arduino 펌웨어

업로드할 파일은 다음 위치입니다.

```text
arduino/CTRL_ZERO_Controller/CTRL_ZERO_Controller.ino
```

Python에서 보내는 직렬 프로토콜은 한 줄 텍스트입니다.

```text
steer,speed\n
```

예시는 다음과 같습니다.

```text
0,45
80,35
-80,35
0,0
```

## 기본 핀 매핑

```text
Motor 1: IN1=2, IN2=3
Motor 2: IN1=4, IN2=5
Steer  : IN1=6, IN2=7
```

방향이 반대로 움직이면 `.ino` 상단의 반전 값을 바꿉니다.

```cpp
const bool MOTOR1_INVERTED = true;
const bool MOTOR2_INVERTED = true;
const bool STEER_INVERTED = false;
```

주의: 현재 펌웨어는 H-bridge 입력핀에 직접 PWM을 주는 방식입니다. 사용하는 모터 드라이버가 ENA/ENB PWM 핀을 따로 요구한다면, 핀 매핑과 `hBridgeForward`, `hBridgeBackward` 구현을 드라이버에 맞게 바꿔야 합니다.

## LiDAR

LiDAR를 쓰려면 `main.py`에서 다음 값을 설정합니다.

```python
USE_LIDAR = True
LIDAR_PORT = "COM5"
```

전방 각도는 기본적으로 330도에서 30도까지입니다. 센서 장착 방향이 다르면 아래 값을 조정합니다.

```python
LIDAR_FRONT_MIN_ANGLE_DEG = 330.0
LIDAR_FRONT_MAX_ANGLE_DEG = 30.0
```

장애물 정지/감속 기준은 다음 값으로 조정합니다.

```python
LIDAR_STOP_DISTANCE_MM = 450.0
LIDAR_SLOW_DISTANCE_MM = 900.0
```
